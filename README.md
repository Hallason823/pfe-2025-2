# PFE 2025-2

## Sobre o Projeto

Repositório de exercícios e trabalhos da disciplina de Programação Front-End (PFE), segundo semestre de 2025. O conteúdo abrange desenvolvimento web moderno com React, Next.js e tecnologias relacionadas.

## Tecnologias Utilizadas

JavaScript | React | Next.js | CSS | Vercel


## Estrutura do Projeto

### `/ex01-next` - Exercícios Next.js
Exercícios práticos focados no desenvolvimento com Next.js:
- Componentes React (funcionais e classes)
- Hooks (useState, useEffect, useContext, useReducer)
- Roteamento com App Router / Pages Router
- Server Components e Client Components
- Renderização Server-Side (SSR) e Static Site Generation (SSG)
- API Routes e integração com backend
- Estilização com CSS Modules e Tailwind CSS
- Gerenciamento de estado

### `/homework` - Trabalhos
Atividades avaliativas que integram múltiplos conceitos de desenvolvimento front-end:
- Desenvolvimento de aplicações web completas
- Integração com APIs externas
- Manipulação de formulários e validação
- Autenticação e autorização
- Otimização de performance
- Responsividade e design adaptativo
- SEO e acessibilidade

## Como Executar

### Pré-requisitos
- Node.js 18 ou superior
- npm ou yarn

### Execução Local

```bash
# Instalar dependências
npm install

# Executar em modo de desenvolvimento
npm run dev

# Build para produção
npm run build

# Executar versão de produção
npm start
```
